# 포스트모템 (에러 사전)

> 에러를 해결한 세션이 즉시 기록한다.
> 반복되는 패턴은 tasks/lessons.md에 한 줄 규칙으로 승격시킨다.

## 파일 구조
- docker.md - 빌드/컨테이너 관련 에러
- db.md - DB/ORM 관련 에러
- frontend.md - 프론트엔드 에러
- infra.md - 인프라/배포 관련 에러

## 기록 형식

### PM-{카테고리}-{번호}: {에러 제목}
- 발생일: YYYY-MM-DD
- 증상: {에러 메시지 또는 현상}
- 원인: {근본 원인}
- 해결: {해결 방법}
- 예방: {재발 방지 규칙}
