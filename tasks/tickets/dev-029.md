  - 멀티라인이 필요한 리스트형 버튼(가이드 아코디언·갱신 배너 토글)은 기존처럼 줄바꿈된다

## 원인
한글(CJK)은 글자 단위로 줄바꿈 가능 → 요소의 min-content 너비가 "1글자". 좁은 flex 컨테이너나
min-w-full 테이블의 액션 셀에서 버튼이 눌리면 라벨이 세로로 1글자씩 출력된다.
개별 버튼마다 whitespace-nowrap 을 붙이는 방식은 60+ 파일에 산재해 누락이 반복됨.

## 수정 (올바른 고도)
- globals.css @layer base: `button { white-space: nowrap }` — 버튼 텍스트는 기본 줄바꿈 금지.
  앞으로 추가되는 버튼도 자동 커버. 멀티라인이 필요한 버튼만 whitespace-normal 로 opt-out
  (utilities 레이어가 base 를 이기므로 기존 truncate/whitespace-* 유틸과 충돌 없음).
- opt-out 2곳: guide 아코디언(제목+부제), licenses 갱신 배너 토글(긴 문장).
- 전역 규칙 비대상인 <Link>/<a> 액션 요소에 개별 nowrap: employees 상세 Link·신규 Link,
  groups 수정 Link, licenses 헤더 신규/Excel.
- notification-bell·global-search 리스트형 버튼은 내부가 truncate 설계라 영향 없음(확인).
