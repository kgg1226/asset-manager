## 배경 — 사용자 보고 (v1.63.0)
"네트워크 자산만 표기되어 편리했던 기능이 전체 하드웨어 노드가 나온다."
dev-062 가 "저장 배치 전체"를 모든 필터 뷰에 올리면서(재진입 소실 방지 목적) 필터 뷰의 가치가 죽은 회귀.

## 뷰별 노드 정책 (확정)
- all: 저장 배치 전체 (페이지 캔버스)
- network: 해당 유형 연결 ∪ 세션 수동 배치 ∪ 네트워크 장비(deviceType=Network)
- pii: 연결 ∪ 세션 수동 배치 ∪ (저장 배치 중 piiStage 지정 자산 — 흐름도 대상만)
- data_flow: 연결 ∪ 세션 수동 배치
작성 경로 보존: 필터 뷰에서 팔레트 배치는 세션 유지, 그 뷰의 연결을 만들거나 piiStage 를 지정하면 영속 표시.
